## Variables

### ES5

```javascript
var name = "Fenil Patel";
```

### ES6+

```javascript
let userName = "ifenilpatel";
const fullName = "Fenil Patel";
```

## Data Types

### Primitive

```javascript
let str = "Hi"; // String
let num = 101; // Number
let bool = true; // Boolean
let undf; // Undefined
let n = null; // Null
let sys = Symbol("symbol"); // Symbol
```

### Non-primitive

```javascript
let array = ["abc", "def", "ghi"]; // Array
let obj = { firstName: "Fenil", lastName: "Patel" }; // Object
```
